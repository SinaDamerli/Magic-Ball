# Magic Ball

This is a simple "Magic Ball" where the user asks a yes or no type of question, and the magic ball provides a random answer. The answers are often surprisingly accurate, making each interaction interesting.

## Table of Contents
1. [Installation]
2. [Usage]
3. [Features]

---

## Installation

No installation is required. The project runs directly in your web browser.

### Prerequisites

- A modern web browser (Google Chrome is recommended).

## Usage

1. Open the `index.html` file in your browser.
2. In the provided input field, type a **yes or no** question.
3. Submit your question and wait for the magic ball to provide its answer.
4. The answer is randomly generated and often accurate.
5. You can ask as many yes or no type questions as you want!

## Features

- **Random Answer Generator**: The magic ball provides random answers to the user's questions.
- **User Interaction**: Users can ask any yes or no type question.
- **Unpredictable Responses**: Each response is randomly chosen from a set of possible answers.
- **Unlimited Questions**: You can ask as many questions as you like.
